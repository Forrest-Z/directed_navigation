# directed_navigation meta-package

ROS meta-package for vehicle navigation considering movement direction bias.

## Packages

### directed_layer

Costmap_2d layer for use with directed_planner.

### directed_map_server

Directed map server.

### directed_msgs

ROS massage definiton.

### direceted_planner

Global planner.
